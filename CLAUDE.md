## Commands

```bash
npm install          # Install dependencies
npm run login        # Authenticate with GitHub via device flow (saves token to public/config.json)
npm start            # Start the interactive CLI REPL
npm run lint         # Run ESLint
```

Pre-commit hooks run automatically on commit: ESLint.

## Architecture

This is a Node.js CLI (mini-copilot) that provides an interactive terminal REPL for chatting with GitHub Copilot.

**Authentication flow:**
1. `scripts/login.js` runs GitHub OAuth Device Flow, saves the GitHub token to `public/config.json`
2. On startup, `src/mini-copilot.js` reads the GitHub token from `public/config.json`, then exchanges it for a short-lived Copilot token via `https://api.github.com/copilot_internal/v2/token`
3. The Copilot token is refreshed every ~24 minutes during the session

**API endpoints:**
- `https://github.com/login/device/code` — Device authorization
- `https://github.com/login/oauth/access_token` — Token polling
- `https://api.github.com/copilot_internal/v2/token` — Copilot token exchange
- `https://api.githubcopilot.com/chat/completions` — Chat completions (GPT-4o model)

**Key files:**
- `src/mini-copilot.js` — Interactive REPL: loads token, maintains conversation history, calls Copilot API
- `scripts/login.js` — CLI login utility (Node.js)
- `public/config.json` — Generated by login script, gitignored; contains `{ github_token, created_at }`
